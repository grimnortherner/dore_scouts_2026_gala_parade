# Dore Scouts 2026 Gala Parade

A simple static web page with the parade instructions for the 267th Dore Scouts Gala Parade & Official Gala Opening Ceremony, 2026.

Published via GitHub Pages. Source content: `index.html` (with the route/assembly map at `assets/parade-route-map.png`).

## Publishing

This repo deploys automatically to GitHub Pages via the workflow in `.github/workflows/pages.yml` whenever `main` is updated. To enable it (one-time, in the repo settings):

1. Go to **Settings → Pages**.
2. Under **Build and deployment → Source**, choose **GitHub Actions**.

After that, every push to `main` publishes the latest page.
